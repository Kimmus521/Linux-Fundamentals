# Linux-Fundamentals Portfolio

A hands-on portfolio demonstrating foundational Linux system administration, shell usage, networking, permissions management, automation, and server operations. This repository documents practical exercises, scripts, configurations, and mini-projects developed while studying Linux fundamentals for cloud engineering and infrastructure technology.

---

## About This Repository

This portfolio was created to strengthen practical Linux skills commonly used in:

- Cloud Engineering
- DevOps
- Site Reliability Engineering (SRE)
- Cybersecurity
- Network Infrastructure
- Systems Administration

The projects and exercises in this repository focus on real command-line workflows and operational tasks rather than theory alone.
